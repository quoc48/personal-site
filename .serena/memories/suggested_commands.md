# Suggested Commands

## Development
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Git Workflow
```bash
# Check status
git status

# Stage and commit
git add <files>
git commit -m "message"

# Push to deploy (triggers Netlify)
git push origin main
```

## System Commands (macOS/Darwin)
```bash
# List files
ls -la

# Find files
find . -name "*.astro"

# Search in files
grep -r "pattern" src/
```

## Astro CLI
```bash
# Run Astro commands
npm run astro -- <command>
```
